# Social-to-Lead Conversational Agent

This is a simple conversational agent I built for a SaaS product (AutoStream).

It can answer user queries like pricing and refund policy, and also detect when a user is interested and collect their details.

## Features

* Handles basic user queries
* Shows pricing from a JSON file
* Handles refund/policy questions
* Captures lead details (name, email, platform)

## How to run

```bash
py main.py
```

## Example inputs

* hi
* price
* refund
* i want to try

## Tech used

* Python
* JSON
